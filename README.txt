Pour utiliser efficacement Bootstrap, vous devez déjà être convaincu de son utilité, vous devez aussi savoir l'installer.
Ce chapitre est destiné à vous faire découvrir en quoi Bootstrap peut vous aider et comment vous devez le mettre en œuvre
pour l'utiliser efficacement. Nous aborderons également les Medias Queries que Bootstrap utilise pour que les pages
s'adaptent automatiquement au support utilisé pour les visualiser.
